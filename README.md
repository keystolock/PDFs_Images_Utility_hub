## PDFs_Images_Utility_hub
A high-performance, free,  web utility tool.* It is used to manipulate all your document, image, and text processing. All file manipulations happen entirely inside your browser and your device.

---

## ✨ Features

### 📄 PDF Utilities
* **Merge & Split PDFs:** Combine multiple PDF documents or extract specific page ranges (e.g., 1–3, 5).
* **Convert:** Convert images (PNG, JPG, WebP) to PDF and extract pages as images.
* **Compression:** Reduce PDF file sizes according to target limits.

### 🖼️ Image & Photo Tools
* **Resizer:** Change Images dimensions in **Pixels (px), Centimeters (cm), Millimeters (mm), or Inches (in)**.
* **Document Presets:** Change Images dimension for **Passport Photos (3.5x4.5cm)**, **Signatures**, and **Government ID forms**.
* **Background Removal & Fill:** Cleanly isolate subjects and remove background or fill with solid colors.
* **Interactive Cropper:** Full drag-and-drop cropping optimized for both desktop mouse and mobile touch controls.
* **Watermark Remover:** White-strip masking with customizable selection bounding boxes.

### 🔤 Text Utility Suite
* **Image & PDF OCR:** Extract text directly from images and multi-page PDF documents.
* **Age Calculator & QR Code Generator:** Fast Age, Cgpa, QR Code Generator.

## 🚀 Getting Started

### Prerequisites
* Node.js 18.0 or higher
* npm or pnpm

### Installation

1. **Clone the repository**
2. **Install npm**
3. **Run npm dev run**
